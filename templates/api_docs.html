Bắt đầu sử dụng

API cho phép truy xuất cookie Netflix đã lưu trong kho lưu trữ Supabase, lấy link đăng nhập tự động, và đăng nhập trên thiết bị TV thông qua mã code.

📋 Luồng sử dụng

1 GET /api/public/list
→
2 Chọn cookie (lấy ID)
→
3 POST get-login-link hoặc tv-login
Base URL: Tất cả endpoint đều bắt đầu với domain server của bạn, ví dụ: https://your-app.onrender.com
Bảo mật: Cookie text gốc không bao giờ được trả ra ở endpoint /api/public/list. Chỉ trả metadata (email, gói, quốc gia...) để đảm bảo an toàn.
Endpoint 1

Danh sách cookie trong kho

Lấy toàn bộ danh sách cookie đã lưu trữ. Kết quả sắp xếp theo thời gian lưu (mới nhất trước). Không bao gồm nội dung cookie gốc.

GET /api/public/list
Response thành công 200 OK 
[
  {
    "id": "3aa4efa7-f20b-4973-8f3a-b9ffb60c308e",
    "savedAt": "2026-04-11T07:53:03.797Z",
    "email": "user@gmail.com",
    "owner": "Nguyễn Văn A",
    "plan": "Premium",
    "country": "US",
    "billing": "15/05/2026",
    "profiles": "User1, User2, Kids",
    "numProfiles": 3,
    "videoQuality": "Ultra HD",
    "maxStreams": "4",
    "login_link": "https://netflix.com/?nftoken=...",
    "nftoken": "abc123..."
  }
]
curl "https://your-server.com/api/public/list"
const res = await fetch('/api/public/list');
const cookies = await res.json();

cookies.forEach(item => {
  console.log(item.id, item.email, item.plan);
});
import requests

res = requests.get("https://your-server.com/api/public/list")
cookies = res.json()

for item in cookies:
    print(item["id"], item["email"], item["plan"])
Endpoint 2

Lấy link đăng nhập

Tạo link đăng nhập Netflix tự động (nftoken) từ cookie đã lưu trong kho. Link mới sẽ được cập nhật lại vào kho lưu trữ tự động.

POST /api/public/get-login-link

Request Body (JSON)

Tham sốKiểuMô tả
id BẮT BUỘC string (uuid) ID của cookie trong kho lưu trữ (lấy từ /api/public/list)
Response thành công 200 OK 
{
  "status": "SUCCESS",
  "login_link": "https://netflix.com/?nftoken=abc123...",
  "nftoken": "abc123...",
  "account": {
    "email": "user@gmail.com",
    "plan": "Premium",
    "country": "US",
    "owner": "Nguyễn Văn A",
    "profiles": "User1, User2",
    "numProfiles": 2
  }
}
Response thất bại 400 / 404 
{
  "status": "ERROR",
  "error": "Cookie hết hạn hoặc không hợp lệ"
}
curl -X POST "https://your-server.com/api/public/get-login-link" \
  -H "Content-Type: application/json" \
  -d '{"id": "3aa4efa7-f20b-4973-8f3a-b9ffb60c308e"}'
const res = await fetch('/api/public/get-login-link', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ id: '3aa4efa7-f20b-...' })
});
const data = await res.json();

if (data.status === 'SUCCESS') {
  console.log('Link:', data.login_link);
  // Mở link trong trình duyệt để đăng nhập tự động
  window.open(data.login_link);
}
import requests

res = requests.post("https://your-server.com/api/public/get-login-link", json={
    "id": "3aa4efa7-f20b-4973-8f3a-b9ffb60c308e"
})
data = res.json()

if data["status"] == "SUCCESS":
    print("Link đăng nhập:", data["login_link"])
Endpoint 3

Đăng nhập TV

Đăng nhập Netflix trên Smart TV / thiết bị streaming bằng mã code hiển thị trên TV, sử dụng cookie đã lưu trong kho.

POST /api/public/tv-login

Request Body (JSON)

Tham sốKiểuMô tả
id BẮT BUỘC string (uuid) ID của cookie trong kho lưu trữ
tv_code BẮT BUỘC string Mã code 8 ký tự hiển thị trên màn hình TV (ví dụ: ABCD1234)
Response thành công 200 OK 
{
  "status": "SUCCESS",
  "message": "Đăng nhập TV thành công!",
  "account": {
    "email": "user@gmail.com",
    "plan": "Premium",
    "country": "US",
    "owner": "Nguyễn Văn A"
  }
}
Response thất bại 
// Trường hợp cookie hết hạn
{ "status": "ERROR", "error": "Cookie DEAD." }

// Trường hợp thiếu authURL
{ "status": "ERROR", "error": "Không tìm thấy authURL..." }

// Trường hợp mã TV sai hoặc hết hạn
{ "status": "FAILED", "message": "Mã code không hợp lệ" }
curl -X POST "https://your-server.com/api/public/tv-login" \
  -H "Content-Type: application/json" \
  -d '{"id": "3aa4efa7-...", "tv_code": "ABCD1234"}'
const res = await fetch('/api/public/tv-login', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    id: '3aa4efa7-f20b-...',
    tv_code: 'ABCD1234'
  })
});
const data = await res.json();

if (data.status === 'SUCCESS') {
  alert('Đăng nhập TV thành công! ' + data.message);
} else {
  alert('Lỗi: ' + (data.error || data.message));
}
import requests

res = requests.post("https://your-server.com/api/public/tv-login", json={
    "id": "3aa4efa7-f20b-4973-8f3a-b9ffb60c308e",
    "tv_code": "ABCD1234"
})
data = res.json()

if data["status"] == "SUCCESS":
    print("✅", data["message"])
else:
    print("❌", data.get("error", data.get("message")))
